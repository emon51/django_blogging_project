# django_blogging_project

## Motivation
 Developed a personal blogging website.

## Features
- Only admin can create, edit and delete blog posts.
- Visitors can read all public blog posts.
  
    

- **Security:**
  - Admin-only access to manage content and the site.



## Used Technologies

- Language: Python
- Backend: Django
- Frontend: HTML, CSS
- Database: Sqlite3
